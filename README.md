# Algoritmux Corporate

Site institucional estatico da Algoritmux, desenvolvido com HTML, CSS e JavaScript puro.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Python, apenas para scripts auxiliares

## Como executar localmente

Este projeto nao possui dependencias de Node.js.

Na raiz do projeto, execute:

```bash
python -m http.server 8000
```

Acesse:

```text
http://localhost:8000
```
